# daily_charge

## 一个使用 flutter 技术开发记账类 App

鉴于当前 app 软件市场中的记账类 app 存在广告推送及可能的存在个人隐私信息泄露问题

本项目 app 中记录的日常账单信息均保存于手机本地，避免信息泄露

本项目功能依然处于功能开发与完善中
